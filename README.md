# nood
nood 是基于spring cloud的微服务分析系统。

## 1. nood 包含的服务
|     服务         |  作用          | 
| --------------- |----------------|
|nood-center      | 注册中心        | 
|nood-config      | 配置中心        | 
|nood-gateway     | 路由网关        |
|nood-auth        | 权限服务        |
|nood-code        | 基础代码        |
|nood-web         | web前端         |
